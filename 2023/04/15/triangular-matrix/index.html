Definition:

A $K \times K$ matrix L is a lower triangular matrix if $L_{i,j}=0$ whenever $i<j$.

A $K \times K$ matrix L is an upper triangular matrix if $L_{i,j}=0$ whenever $i>j$.

Proposition 1:

The transpose of a triangular matrix is also triangular.

Proposition 2:

the product of two triangular matrices is triangular.

Proposition 3:

A triangular matrix is invertible if and only if its diagonal entries are non-zero.

Proof (part 1):

If the diagonal entries of a lower-triangular matrix are non-zero, then it is invertible.

Proof (part 2):

A lower-triangular matrix $L$ is invertible if its diagonal entries are non-zero.

Suppose the $k^{th}$ entry of the diagonal of matrix $L$ is zero.

Consider a $k \times k$ matrix that is formed by the first $k$ rows of $L$.

this matrix has at most $k-1$ independent rows.

As a consequence, the $k$ rows of the matrix are not independent.

$L$ is not full-ranked and is not invertible.
